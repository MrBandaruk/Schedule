# Schedule

>  ##### Schedule is an ASP.NET MVC 5 Web-Application.

- **News**: 
  - CRUD
  - Paging
  - Sorting
  - Searching

- **News Panel**:
  - CRUD
  - jQuery DataTables
  - jQuery slick
  - Paging
  - Sorting
  - Searching

- **Article**:
  - jQuery slick

- **Calendar**:
  - CRUD
  - jQuery fullCalendar + jQuery Scheduler
  - jQuery dataTables + moment.js

- **Authentication**:
  - ASP.NET Identity
